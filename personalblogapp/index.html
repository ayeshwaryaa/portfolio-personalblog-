Social Media Analyzer

The "Social Media Analyzer App" is a Java project with a user-friendly GUI built using Java Swing to analyze engagement metrics like likes, comments, shares, and posts. It uses a `Post` class for data and a `SocialMediaAnalyzer` class for computing statistics. Users input post data, and the app dynamically displays metrics with a "Refresh" button. The project demonstrates OOP principles and Java's capability for creating interactive applications, helping users efficiently track social media interactions.
